# Kalkulačka
Tato softwarová aplikace je kalkulačka, která umožňuje provádět základní matematické operace, jako jsou sčítání, odčítání, násobení, dělení a funkce mocniny, odmocniny, přirozeného logaritmu a faktoriálu.
