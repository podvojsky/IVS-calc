Komunikace v týmu
-----------------

- Jako komunikační kanál byl zvolen privátní Discord server.
- Server je rozdělen na jednotlivé místnosti, které korespondují
  s různými částmi projektu.
- Každý člen týmu bude aktivně server sledovat (jednou denně).
- V případě konkrétního dotazu na člena týmu se využije tzv. ping, který člena upozorní.
- Pokud to bude nutné uskuteční se videohovor a probere se další postup.

Verzovací systém
----------------

- Jako verzovací systém byl zvolen program git
  se vzdáleným hostingem na serveru Github.
- V projektu bude využita funkcionalita Githubu
  (Github Projects, Issues, Reviews, Pull Requests).
- Repositář projektu je privátní.
- Do repositáře mají přístup pouze členové týmu
  a účet "ivskontrola" pro možnost náhledu do projektu vyučujícími/cvičícími
- Repositář je dostupný zde: https://github.com/podvoj/IVS-calc
